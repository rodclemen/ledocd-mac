User manual

Make your LEDs look like they belong in a pinball machine.

LED OCD for Mac lets you configure your installed LED OCD board: assign lamps to brightness profiles, fine-tune the fade, and permanently save the result to the board.

LED OCD hardware and the original LED OCD Windows software were created by Harold at ledocd.com. This Mac manual is an independent companion guide and draws on the established LED OCD workflow and terminology.

Part 1

What the app does

Your pinball machine decides when a lamp is on. LED OCD changes how that lamp looks when it turns on: brightness, fade, and the character of the transition.

01

The pinball machine

Still controls the game and tells each lamp when to switch on or off.

02

The LED OCD board

Sits in the machine and applies your brightness and fading choices to the lamps.

03

The Mac app

Connects to the board, lets you edit its configuration, and helps you save a backup.

Think of a profile as a brightness recipe

There are eight profiles. Each profile has a name, a dimmest level, a brightest level, and a fade delay. You assign lamps to a profile; lamps sharing a profile share the same behavior.

For example, you might use one profile for soft inserts, another for bright inserts, and a third for lamps that need a slower fade. You are not programming the machine. You are deciding how its LEDs should behave.

Important before you begin

Game Select gives you lamp names — not preset lamp settings.

This distinction matters. Choosing a game makes the lamp table understandable; it does not load somebody else’s brightness setup onto your board.

Why no brightness or profile values are stored with each game?

The game files contain the lamp number and lamp name, such as Left Ramp or Jackpot. They do not include brightness, fade, or profile assignments. Those settings depend heavily on the LEDs installed in the machine: LED colour, make, diffuser, brightness, and even how a particular game’s inserts look. A setting that looks perfect with one LED set may look terrible with another.

For that reason, LED OCD does not pretend there is one correct configuration for every machine. You create the settings that suit your LEDs and your machine.

What Game Select does

Shows the proper lamp numbers and friendly insert names, so you know exactly which lamp you are assigning to a profile. It also selects the correct lamp layout for the machine type.

What it does not do

It does not reset your current brightness or profile values, and it does not retrieve a ready-made LED configuration. Selecting a different game simply changes the lamp names shown in the table.

Keep one export file per machine

When you have tuned a machine to your liking, use Export. That file contains your eight profiles and your lamp-to-profile assignments. Later, use Import to restore the configuration, then Send and Save it to the board.

Part 2

Change lamp settings and store them on the board

This is the normal workflow for an installed LED OCD board. You can fine-tune as much as you like, but the basic routine stays pleasantly boring.

Connect→Choose game→Assign profiles→Adjust→Send→Save
  1. Turn on the pinball machine and connect USB.

    The LED OCD board must already be installed correctly. Connect its USB cable to the Mac and open LED OCD for Mac.

  2. Click Scan, choose the board, then click Connect.

    Select the USB serial connection, normally named something similar to cu.usbserial-…. The app identifies the board and automatically reads the settings currently stored on it. A green Connect button means the connection worked.

  3. Select your machine in Game Select.

    The lamp table now shows useful names for the lamps in that game. Select the correct machine type before reading or sending settings, especially when moving between WPC, Stern, or Capcom machines.

  4. Find the lamps you want to tune and choose their profile.

    In the lamp table, set the Profile number for each lamp. Several lamps can use the same profile, which makes consistent tuning much easier.

  5. Adjust the profile itself.

    Set B1 for the dimmest level, B8 for the brightest level, and Delay for fade speed. Leave Advanced off at first; the app will calculate the middle brightness steps automatically.

  6. Click Send and inspect the playfield.

    Send places the new configuration on the board so you can see the result immediately. A green Send button means there are changes in the app that the board has not yet received.

  7. Repeat until it looks right.

    Make small changes, Send again, and compare. For faster hands-on tuning, use Live Mode.

  8. Click Save.

    Save writes the active configuration into the board’s permanent memory. A red Save button means you have sent changes that will be lost after power-off unless you save them.

  9. Export a backup.

    Once you are happy with the result, Export a copy to your Mac. The board now has your permanent settings, and the Mac has a recoverable backup.

Optional, but recommended

Live Mode: tune while watching the actual lamps

Live Mode is the best way to judge brightness and fade. The app sends profile edits to the board immediately, so you can adjust a value while looking at the playfield.

How to use it

Connect, select the correct game, then open Live Mode. Press the bulb button beside a profile to light every lamp that uses it. Change B1, B8, and Delay while watching the lamps.

When you are done

Click Commit Changes. In Live Mode, this single action both sends the changes and permanently saves them. Leaving Live Mode returns lamp control to normal play.

Do not forget to commit.

Live Mode makes changes visible immediately, but visible is not the same as permanently saved. Commit Changes is red while there are edits that have not yet been stored on the board.

Profile controls

B1 — minimum brightness

The first and dimmest level of the brightness ramp. Lower B1 makes the start gentler; higher B1 makes lamps look brighter sooner.

B8 — maximum brightness

The final and brightest level. Lower it when LEDs are harsh or overpowering; raise it when they are too subdued.

Delay

Controls how quickly the brightness steps change. Lower values are faster; higher values create a slower, smoother fade.

B2 through B7

The intermediate steps. The app calculates these automatically unless Advanced is switched on.

Keep your work

Import, Export, and the three places settings can live

There is no hidden Mac database keeping your lamp setup. This is deliberate: the board is the working home; export files are your portable backups.

BOARD

Permanent machine settings

Settings stay here only after Save or Commit Changes. These are the settings your pinball machine actually uses.

APP

Temporary working copy

The app holds the current edit while it is open. Quit without exporting and there is no Mac-side copy to recover later.

EXPORT FILE

Your backup and transfer file

Export saves the current profiles and lamp assignments as an XML file that can be imported later, including in the Windows software.

Recommended habit

Use a clear filename after every successful tuning session, for example Attack_from_Mars_LED_setup_2026-07.xml. It is cheap insurance. A good configuration takes longer to recreate than to export.

After importing a file, use Send and then Save. Import loads the file into the app; it does not automatically write it to the board.

Common questions

Things that often look like problems

I selected my game, but the brightness settings did not change.

That is expected. Game Select supplies lamp names and the correct lamp layout, not an LED preset. Import an exported configuration if you want to load saved tuning values.

I sent changes, then turned the machine off. They are gone.

Send is only a live transfer. Use Save after Send, or Commit Changes in Live Mode, to store the configuration permanently.

The wrong lamps seem to have received the settings.

Check that the correct game type is selected. If you changed Game Select after connecting, click Read again so the board configuration is decoded using the correct lamp layout.

I cannot connect to the board.

Click Scan again, select the USB serial device, check the USB cable, and look in the Log panel for the exact connection error.

Everything is too complicated. Where should I start?

Leave Advanced off. Use one profile for dim inserts, one for normal inserts, and one for bright inserts. Tune B1, B8, and Delay in Live Mode first.

Can I experiment safely?

Yes. Export your known-good configuration first. You can then experiment, re-import the backup, Send, and Save to return to the working setup.
