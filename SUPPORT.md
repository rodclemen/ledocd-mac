# Support

## LED OCD for Mac

Use [GitHub Issues](https://github.com/rodclemen/ledocd-mac/issues) for:

- reproducible app bugs;
- feature requests;
- build problems;
- questions about using the macOS app; and
- compatibility problems involving the app.

Before opening an issue, check the README, the built-in manual, and existing
issues.

When reporting a problem, include the app version, macOS version, hardware
details, and clear reproduction steps. Remove credentials and personal
information from logs and screenshots.

## Security reports

Do not report suspected security vulnerabilities in a public issue. Follow
[`SECURITY.md`](SECURITY.md).

## Original hardware and firmware

This repository is an independent app project. It is not the official support
channel for LED OCD or GI OCD hardware, original firmware, orders, warranties,
or repairs.

For those matters, consult the official LED OCD documentation or the current
hardware supplier.
