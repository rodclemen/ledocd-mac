# Security policy

## Supported versions

Security corrections are normally made only in the latest released version of
LED OCD for Mac.

| Version | Supported |
| --- | --- |
| Latest release | Yes |
| Older releases | No |

## Reporting a vulnerability

Do not report suspected vulnerabilities in a public GitHub issue.

Use GitHub's private vulnerability reporting for this repository:

1. Open the repository's **Security** tab.
2. Open **Advisories**.
3. Select **Report a vulnerability**.
4. Include the affected version, reproduction steps, potential impact, and any
   suggested mitigation.

Do not include real passwords, access tokens, private keys, Apple notarization
credentials, personal information, or other secrets in a report.

Please allow a reasonable opportunity to investigate and release a correction
before publishing vulnerability details.
